◆ CameraGetHardwareMirror()

MVSDK_API CameraSdkStatus __stdcall CameraGetHardwareMirror ( CameraHandle  hCamera,
int  iDir,
BOOL *  pbEnable 
)

Get the hardware mirrored state of the image.

Parameters
[in]hCameraCamera handle.
[in]iDirIndicates the mirroring direction to be obtained. 0 means horizontal direction; 1 means vertical direction.
[out]pbEnableReturns TRUE, indicating that the direction mirror image of iDir is enabled.
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.

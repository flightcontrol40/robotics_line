◆ CameraImageProcess()

MVSDK_API CameraSdkStatus __stdcall CameraImageProcess ( CameraHandle  hCamera,
BYTE *  pbyIn,
BYTE *  pbyOut,
tSdkFrameHead *  pFrInfo 
)

The obtained raw camera output image data is processed to superimpose processing effects such as saturation, color gain and correction, noise reduction, etc. Finally, image data of RGB888 format is obtained.

Parameters
[in]hCameraCamera handle.
[in]pbyInThe buffer address of the input image data cannot be NULL.
[out]pbyOutThe buffer address of the image output after processing cannot be NULL.
[in,out]pFrInfoAfter inputting the frame header information of the image, the image format uiMediaType in the frame header information will be changed after the processing is completed.
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.
Examples:
grab.cpp, roi.cpp, soft_trigger.cpp, win_basic.cpp, and win_callback.cpp.

◆ CameraImageProcessEx()

MVSDK_API CameraSdkStatus __stdcall CameraImageProcessEx ( CameraHandle  hCamera,
BYTE *  pbyIn,
BYTE *  pbyOut,
tSdkFrameHead *  pFrInfo,
UINT  uOutFormat,
UINT  uReserved 
)

The obtained raw camera output image data is processed to superimpose processing effects such as saturation, color gain and correction, noise reduction, etc. Finally, image data of RGB888 format is obtained.

Parameters
[in]hCameraCamera handle.
[in]pbyInThe buffer address of the input image data cannot be NULL.
[out]pbyOutThe buffer address of the image output after processing cannot be NULL.
[in,out]pFrInfoAfter inputting the frame header information of the image, the image format uiMediaType in the frame header information will be changed after the processing is completed.
[in]uOutFormatThe output format of the image after processing. It may be one of CAMERA_MEDIA_TYPE_MONO8,CAMERA_MEDIA_TYPE_RGB,CAMERA_MEDIA_TYPE_RGBA8.
[in]uReservedReservation parameters must be set to 0.
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.

◆ CameraAlignMalloc()

MVSDK_API BYTE* __stdcall CameraAlignMalloc ( int  size,
int  align 
)

Apply for an aligned memory space. The function is similar to malloc, but the returned memory is aligned with the number of bytes specified by align.

Parameters
[in]sizeSize of the space.
[in]alignThe number of aligned bytes.
Returns
Successful a non-zero value is returned indicating the first address of the memory. Fails to return NULL.
Note
Memory allocated must be freed using CameraAlignFree
Examples:
grab.cpp, roi.cpp, soft_trigger.cpp, win_basic.cpp, and win_callback.cpp.

Set the parameters for the specified crosshairs.

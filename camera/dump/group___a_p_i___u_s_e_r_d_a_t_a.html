◆ CameraWriteSN()

MVSDK_API CameraSdkStatus __stdcall CameraWriteSN ( CameraHandle  hCamera,
BYTE *  pbySN,
INT  iLevel 
)

Set the camera's serial number.

Parameters
[in]hCameraCamera handle.
[in]pbySNThe buffer for the serial number.
[in]iLevelThe serial number to be set can only be 1 or 2.
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.
Note
Our company camera serial number is divided into 3 levels. Level 0 is our company's custom camera serial number, which has been set at the factory and cannot be modified. Levels 1 and 2 are reserved for secondary development. Each serial number length is 32 bytes.

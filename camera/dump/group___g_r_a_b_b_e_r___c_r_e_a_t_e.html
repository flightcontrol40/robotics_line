◆ CameraGrabber_Create()

MVSDK_API CameraSdkStatus __stdcall CameraGrabber_Create ( void **  Grabber,
tSdkCameraDevInfo *  pDevInfo 
)

Create Grabber from device info.

Parameters
[out]Grabberreturns newly created grabber
[in]pDevInfodevice information. CameraEnumerateDevice
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.
Note
This function uses CameraInit internally to open the camera, so you can use CameraGrabber_GetCameraHandle to get the camera handle and use other SDK APIs to operate the camera.
Examples:
win_grabber.cpp.

◆ CameraSetClrTempMode()

MVSDK_API CameraSdkStatus __stdcall CameraSetClrTempMode ( CameraHandle  hCamera,
int  iMode 
)

Color temperature mode used when setting white balance.

Parameters
[in]hCameraCamera handle.
[in]iModemode, can only be defined by emSdkClrTmpMode
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.
Note
There are three supported modes, automatic, preset, and custom.
Automatic mode will automatically select the appropriate color temperature mode
In preset mode, user-specified color temperature mode is used
custom-defined color temperature digital gain and matrix

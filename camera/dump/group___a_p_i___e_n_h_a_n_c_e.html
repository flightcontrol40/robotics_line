◆ CameraGetCurrentLut()

MVSDK_API CameraSdkStatus __stdcall CameraGetCurrentLut ( CameraHandle  hCamera,
int  iChannel,
USHORT *  pLut 
)

Obtain the camera's current LUT table, which can be called in any LUT mode, to intuitively observe changes in the LUT curve.

Parameters
[in]hCameraCamera handle.
[in]iChannelSpecifies the LUT color channel to be obtained. When LUT_CHANNEL_ALL, returns the LUT table of the red channel.
See also
emSdkLutChannel
Parameters
[out]pLutpoints to the address of the LUT table. The LUT table is an unsigned short integer array, and the array size is 4096, which is the mapping value corresponding to the code color channel from 0 to 4096 (12 bit color accuracy).
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.

Set 3D noise reduction parameters.

[in]WeightsNoise reduction weight, such as when using 3 pictures for noise reduction, this parameter can be passed in 3 floating points (0.3, 0.3, 0.4).

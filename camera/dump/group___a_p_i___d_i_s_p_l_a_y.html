◆ CameraDisplayInit()

MVSDK_API CameraSdkStatus __stdcall CameraDisplayInit ( CameraHandle  hCamera,
HWND  hWndDisplay 
)

Initialize the display module inside the SDK. The function must be called before calling CameraDisplayRGB24. If you use your own method for image display (do not call CameraDisplayRGB24) during secondary development, you do not need to call this function.

Parameters
[in]hCameraCamera handle.
[in]hWndDisplayThe handle of the display window, typically the m_hWnd member of the window.
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.
Examples:
win_basic.cpp, and win_callback.cpp.

◆ CameraSetDisplayOffset()

MVSDK_API CameraSdkStatus __stdcall CameraSetDisplayOffset ( CameraHandle  hCamera,
int  iOffsetX,
int  iOffsetY 
)

Set the starting offset of the display. This is valid only when the display mode is DISPLAY MODE_REAL. For example, the size of the display control is 320×240, and the size of the image is 640×480. When iOffsetX = 160 and iOffsetY = 120, the displayed area is the center 320×240 of the image.

Parameters
[in]hCameraCamera handle.
[in]iOffsetXThe offset's X coordinate.
[in]iOffsetYThe offset's Y coordinate.
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.

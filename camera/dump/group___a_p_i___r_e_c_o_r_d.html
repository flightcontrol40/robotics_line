MVSDK_API CameraSdkStatus __stdcall CameraInitRecord ( CameraHandle  hCamera,
int  iFormat,
char *  pcSavePath,
BOOL  b2GLimit,
DWORD  dwQuality,
int  iFrameRate 
)

Initialize a video.

Parameters
[in]hCameraCamera handle.
[in]iFormatEncoding format (0: no compression 1: MSCV compression 4: H264)
[in]pcSavePathThe path to save the video file.
[in]b2GLimitIf TRUE, the file is automatically split when it is larger than 2G. (Function not implemented)
[in]dwQualityThe larger the quality factor of the video, the better the quality. Range 1 to 100.
[in]iFrameRateThe frame rate of the video. It is recommended to set a larger frame rate than the actual acquisition so that no frames are missed.
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.

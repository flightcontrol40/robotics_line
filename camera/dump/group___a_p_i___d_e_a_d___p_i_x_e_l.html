◆ CameraReadDeadPixels()

MVSDK_API CameraSdkStatus __stdcall CameraReadDeadPixels ( CameraHandle  hCamera,
USHORT *  pRows,
USHORT *  pCols,
UINT *  pNumPixel 
)

Reading camera dead pixels.

Parameters
[in]hCameraCamera handle.
[out]pRowsdead pixels y coordinates
[out]pColsbad x coordinate
[out]pNumPixelInputs the size of the row and column buffers. When returned, it indicates the number of bad pixels returned in the row and column buffers.
Returns
Returns CAMERA_STATUS_SUCCESS(0) successfully. Otherwise, it returns a non-zero error code. Please refer to the definition of the error code in CameraStatus.h.
Note
When pRows or pCols is NULL, the function will return the camera's current number of dead pixels through pNumPixel.
